Important FAQ
What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to a part of the document. It allows developers to access, modify, and interact with the content and structure dynamically using languages like JavaScript.

How do you select an element from the DOM?

To select an element from the DOM, you can use document.getElementById('id') to select an element by its ID or document.querySelector('.class') to select the first element that matches a class or tag. For multiple elements, document.querySelectorAll('tag') can be used to select all matching elements.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the DOM is a technique where a single event listener is added to a parent element to handle events for its child elements, taking advantage of event bubbling. It is useful because it improves performance and simplifies code by reducing the number of event listeners needed, especially for dynamically added elements.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes, use element.setAttribute('attribute', 'value') or access properties directly like element.id = 'newId'. For styles, modify them using element.style.property = 'value' or by adding/removing CSS classes with element.classList.
